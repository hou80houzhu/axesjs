工具支持

专为packet精心打造的相关开发工具

packetBuilder
packetBuilder是用于构建基于packet开发的项目的项目构建工具，它依赖于Nodejs环境。它对静态资源进行常规的合并压缩等操作并最终生成用于项目部署的资源表。
github
packetToolkit
packetToolkit是基于chrome的浏览器插件，用于辅助开发基于packet的工程项目。
github
corgiserver
corgiserver是运行于nodejs环境的web服务器，类似java环境的Tomcat。
github
pageMaker
pageMaker是实现定制响应式网站的开发套件（暂时只支持Java环境），包括后端开发支持支持库以及前端UI可视化编辑工具。
github
